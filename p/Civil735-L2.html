Python, by itself, is an open-source programming language with limited capacities

Python replies on the use of "modules"

Pandas is one of the two major data analytics modules in Python, along with Numpy

It is a fundamental high-level building block for doing practical, real world data analysis in Python.

providing fast, flexible, and expressive data structures designed to work with large datasets.

Pandas is about Dataframes (like Excel spreadsheet)

1-dimensional data in Pandas is called Series: array with labels, contains any data type: integers, string, float, Python objects and more

Dataframe is a 2-dimensional data structure with labels
