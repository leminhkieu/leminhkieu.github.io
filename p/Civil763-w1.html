Course modules:

Module 1: Python programming for Engineers (4 weeks – 12 hours)

Module 2: Exploratory Data Analytics (4 weeks – 12 hours)

Module 3: Machine Learning and case studies in Smart Infrastructure Analytics (2 weeks – 6 hours)

The remaining 9 hours are for helpdesk and presentation time

Course learning outcomes:

1. Write Python programming code using popular tools such as Jupyter Notebook

2. Analyse the suitability and limitations of data analytics tools, algorithms, and platforms for real-world data science problems

3. Apply cutting-edge techniques from large-scale data mining, machine learning and statistical analysis through assignments and a final project

4. Apply statistical and geospatial data visualisation techniques to communicate and visually represent their findings to both technical and non-technical audiences

Course assessments:

Data analytics assignment (30%)

Data science final project (40%)

Creative & effective presentation of the final project(10%)

4 data analytics exercises (5% each)

What is data analytics?

The concept is too recent that there is currently no single agreed definition

Data analytics is the science of analysing raw data in order to make conclusions about that information
