Virtual environment

An isolated workspace where specific versions of Python, as well as its libraries and packages

Benefits:

Dependency management and Version control: Each project can have its own dependencies with specific version

Simplicity: It's easy to send someone a list of packages they would need, deleting or adding into a virtual environment

Less risk: isolating your application and its environment

Python packages for data science

A Python package is a collection of related scripts and code modules that are grouped together so they can be easily used and shared.

What is a Jupyter Notebook?

It's a single document where you can run code, display the output, and also add explanations, formulas, charts, and make your work more transparent, understandable, repeatable, and shareable.
