Course modules:

Module 1: Python programming for Engineers (3 weeks – 9 hours)

Module 2: Machine Learning for Smart Infrastructure Systems (3 weeks – 9 hours)

Module 3: Case studies in Smart Infrastructure Analytics (3 weeks – 9 hours)

The remaining 9 hours are for guest lectures, helpdesk and presentation time

Course learning outcomes:

1. Write Python programming code using popular tools such as Jupyter Notebook

2. Analyse the suitability and limitations of data analytics tools, algorithms, and platforms for real-world data science problems

3. Apply cutting-edge techniques from large-scale data mining, machine learning and statistical analysis through assignments and a final project

4. Apply statistical and geospatial data visualisation techniques to communicate and visually represent their findings to both technical and non-technical audiences

Course assessments:

Data analytics assignment (30%)

Data science project (40%)

Creative presentation of the project (10%)

4 quizzes (5% each)
