Learning outcomes of the week:

1. Understand the fundamentals of data visualisations

2. Evaluate the suitability of common types of graphs on a real data

Why data visualisation?

Understand the data

Observe patterns in the data

Define a research problem: what can we do with the data?

Data visualisation is harder than it looks!

Priciples of data visualisation

Maximise data-ink ratio

Minimise lie factor

Data-ink ratio = Info / Total ink used to print the graph

Extraneous visual elements distract from the message the data is trying to tell: minimise chartjunks

Minimise lie factor

Data visualisation needs to be fair and truthful

Some researchers don't like pie charts anyway because it is hard to see the comparison between pieces

Minimise lie factor

Always start bar graphs at zero

Always properly label the axes

Minimise lie factor

Aim for 45 degree lines or Golden ratio as most interpretable
